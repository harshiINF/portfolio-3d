# portfolio-3d
A modern developer portfolio with immersive 3D visuals and smooth interactive animations. Built using React, TypeScript, and modern web technologies to showcase projects, systems engineering experience, and technical skills. Designed with performance, modular architecture, and a futuristic UI to create an engaging developer showcase

Installation
Clone the repository:

git clone https://github.com/yourusername/portfolio3d.git
Go to the project directory:

cd portfolio3d
Install dependencies:

npm install
Start the development server:

npm run dev
The application will run locally at:

http://localhost:5173
